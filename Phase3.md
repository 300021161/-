## Оценка проекта

Фаза 3 посвящена комплексной оценке выбранного проекта "Цифровой стилист с функцией виртуальной примерки" с использованием четырёх независимых методов оценки для определения реальных трудозатрат, сроков и состава команды.
### 1. Оценка COSMIC Function Points
объективная оценка функциональности
### 2. Оценка Use Case Points
оценка через сложность Use Cases
### 3. Коллективная экспертная оценка (Делфи)
коллективная экспертная оценка
### 4. Оценка User Stories через Story Points
Agile-подход через декомпозицию

### Итоговая оценка проекта:

#### Рекомендуемая оценка с запасом:
- Трудозатраты: 900 часов
- Запас: 15 % на непредвиденные риски

#### Рекомендуемая команда:
- Backend-разработчик - 1 человек
- Frontend-разработчик - 1 человек
- ML/AI инженер - 1 человек
- Дизайнер UX/UI - 1 человек
- Аналитик/Менеджер - 1 человек
