# **Оценка проекта**
## **Что было сделано**
Мы провели комплексную оценку проекта "Цифровой стилист" четырьмя разными методами.

## **Методы оценки:**
- COSMIC Function Points — подсчитали все функции приложения
- Use Case Points — оценили сложность пользовательских сценариев
- Экспертная оценка — провели коллективное обсуждение в команде
- User Stories — разбили всё на мелкие задачи и оценили каждую

## **Результаты методов:**
Минимальная оценка: 440 часов
Максимальная оценка: 1490 часов
Средняя реалистичная оценка: около 850 часов

## **Итоговое решение**
Было решено планировать 900 часов работы (с запасом 15% на непредвиденные сложности).

## MVP:
- Регистрация и профиль пользователя
- Базовые рекомендации одежды
- Простая 2D-примерка (без сложной AR)
- Возможность сохранять понравившиеся образы

Сроки MVP - 3 месяца
Бюджет на разработку ~ 1.4 млн рублей
Дальнейшее развитие будет осуществляться по отзывам пользователей

## Не будет в первой версии (оставим на потом):
- Сложная 3D-примерка с дополненной реальностью
- Интеграция с большим количеством магазинов
- Автоматическое обновление гардероба

## Причина:
80% пользы для пользователей дают 20% функций. Лучше выпустить простую версию быстро и улучшать по отзывам. Сложные функции требуют много времени, но не критичны для первых пользователей.

## Главный вывод
Оценка показала, что даже "простой" проект требует серьёзных временных и финансовых затрат. Но разумное планирование MVP позволяет начать быстрее и дешевле, проверяя идеи на реальных пользователях.
